# TeamMEW
Team MEW page
### Installation

Dillinger requires [Node.js](https://nodejs.org/) v9+ to run.

Install the dependencies and devDependencies and start the server.

```sh
$ cd TeamMEW
$ npm install
$ npm run dev
```

For production environments...

```sh
$ npm install
$ npm run build
```
